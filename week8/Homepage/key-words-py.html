Key Words

  • int

    Python uses int to represent integer values.
    For example:

    count = 5

  • for

    The for loop in Python is used to iterate over sequences like lists or ranges.
    For example:

    for i in range(9):
    print(i, end=" ")

  • if and else

    Python uses if and else for decision-making.
    For example:

    if i == 1:
    print("i is 1")
else:
    print("i is not 1")

    If the value of i is other than 1, the output will be:

    i is not 1

  • float

    The float type is used for decimal numbers in Python.
    For example:

    number = 3.14

  • while

    The while loop runs as long as the condition is true.
    For example:

    i = 0
while i < 10:
    print(i, end=" ")
    i += 1

  • const (not a keyword)

    Python does not have a const keyword. Constants are by convention written in uppercase.
    For example:

    PI = 3.14159

  • str

    The str type is used to represent strings.
    For example:

    name = "Alice"

  • match and case

    Python 3.10+ introduced match and case for structural pattern matching.
    For example:

    match value:
    case 1:
        print("One")
    case 5:
        print("Five")
    case _:
        print("Default")

  • break and continue

    break exits the loop immediately. continue skips to the next iteration.
    For example:

    for i in range(1, 11):
    if i == 3:
        continue
        if i == 7:
            break
            print(i, end=" ")

    Output:

    1 2 4 5 6
